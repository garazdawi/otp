[1m  is_function(Term)[0m

  Returns [4mtrue[0m if [4mTerm[0m is a fun; otherwise, returns [4mfalse[0m.

[1mExamples[0m

    1> is_function(fun() -> ok end).
    true
    2> is_function(fun lists:sum/1).
    true
    3> is_function({lists,sum}).
    false
