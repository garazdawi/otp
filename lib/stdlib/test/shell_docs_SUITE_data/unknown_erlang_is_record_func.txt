[1m  is_record(Term)[0m

[4mSince[0m:
  OTP 29.0

  Returns [4mtrue[0m if [4mTerm[0m is a native record; otherwise, returns [4m[0m
  [4mfalse[0m.

[1mExamples[0m

    1> NR = records:create(test, native, [], #{is_exported => true}).
    2> is_record(NR).
    true
    3> is_record(an_atom).
    false
    4> is_record({a,b,c}).
    false

[1m  is_record(Term, Name)[0m

  Returns [4mtrue[0m if [4mTerm[0m is either a tuple record or a native
  record named [4mName[0m; otherwise, returns [4mfalse[0m.

  This is allowed in guard tests if [4mName[0m is a literal atom.

  The compilation will fail with an error message if [4mName[0m does not
  refer to either a record defined in the same module as the call to [4m[0m
  [4mis_record/2[0m or a native record imported into it.

  If [4mName[0m is a variable and [4mTerm[0m is a native record, this
  function returns [4mtrue[0m if [4mName[0m refers to a native record named [4m[0m
  [4mName[0m in any module; otherwise, it returns [4mfalse[0m.

  If [4mName[0m is a variable and [4mTerm[0m is tuple, this function returns [4m[0m
  [4mtrue[0m if the first element of the tuple is [4mName[0m; otherwise, it
  returns [4mfalse[0m.

  [4mChange[0m

    Before OTP 29, this function would only work for tuple
    records.

[1m  is_record(Term, RecordTag, Size)[0m

  Returns [4mtrue[0m either if [4mTerm[0m is a native record defined in
  module [4mModule[0m with name [4mName[0m, or if [4mTerm[0m is a tuple, its
  first element is [4mName[0m, and its size is [4mArity[0m; otherwise,
  returns [4mfalse[0m.

  This is allowed in guard tests if either [4mModule[0m and [4mName[0m are
  literal atoms, or [4mName[0m is a literal atom and [4mSize[0m is a literal
  integer.

  [4mNote[0m

    This BIF only checks if [4mTerm[0m is a native record created from
    the given module. It does not check whether [4mTerm[0m is still
    defined in the given module, nor whether it is exported.

  [4mChange[0m

    Before Erlang/OTP 29, this function only worked for tuple
    records.
