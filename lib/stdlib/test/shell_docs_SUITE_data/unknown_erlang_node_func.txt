[1m  node()[0m

  Returns the name of the local node.

  If the node is not alive, [4mnonode@nohost[0m is returned.

[1mExamples[0m

    > node().
    nonode@nohost

[1m  node(Arg)[0m

  Returns the node where [4mArg[0m originates.

  If [4mArg[0m originates from the local node and the local node is not
  alive, [4mnonode@nohost[0m is returned.

[1mExamples[0m

    1> node(self()) =:= node().
    true
