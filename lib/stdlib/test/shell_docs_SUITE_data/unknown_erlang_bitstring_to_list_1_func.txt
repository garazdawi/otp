[1m  bitstring_to_list(Bitstring)[0m

  Returns a list of integers corresponding to the bytes of [4m[0m
  [4mBitstring[0m.

  If the number of bits in the binary is not a multiple of 8, the
  last element of the list is a bitstring containing the remaining 1
  to 7 bits.

[1mExamples[0m

    1> bitstring_to_list(<<433:16>>).
    [1,177]
    2> bitstring_to_list(<<433:16,3:3>>).
    [1,177,<<3:3>>]
