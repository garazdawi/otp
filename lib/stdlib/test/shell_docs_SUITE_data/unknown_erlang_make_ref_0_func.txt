[1m  make_ref()[0m

  Returns a unique reference.

  The reference is unique among connected nodes.

  [4mWarning[0m

    Before OTP 23, if a node was restarted multiple times with the
    same node name, references created on a newer instance could
    be mistaken for those created on an older instance with the
    same name.

[1mExamples[0m

    1> is_reference(make_ref()).
    true
