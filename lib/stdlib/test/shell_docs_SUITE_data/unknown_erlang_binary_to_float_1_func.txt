[1m  binary_to_float(Binary)[0m

[4mSince[0m:
  OTP R16B

  Returns the float whose text representation is [4mBinary[0m.

  The float string format is the same as the format for Erlang float
  literals, except that underscores are not permitted.

  Failure: [4mbadarg[0m if [4mBinary[0m contains an invalid representation
  of a float.

[1mExamples[0m

    1> binary_to_float(~"10.5").
    10.5
    2> binary_to_float(~"17.0").
    17.0
    3> binary_to_float(<<"2.2017764e+1">>).
    22.017764
