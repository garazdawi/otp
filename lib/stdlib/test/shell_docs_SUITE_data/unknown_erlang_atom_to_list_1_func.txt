[1m  atom_to_list(Atom)[0m

  Returns a list of unicode code points corresponding to the text
  representation of [4mAtom[0m.

  See the [4municode[0m module for instructions on converting the
  resulting list into different formats.

[1mExamples[0m

    1> atom_to_list('Erlang').
    "Erlang"
    2> atom_to_list('π').
    [960]
    3> atom_to_list('你好').
    [20320,22909]
