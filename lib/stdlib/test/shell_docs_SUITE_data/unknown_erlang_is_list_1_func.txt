[1m  is_list(Term)[0m

  Returns [4mtrue[0m if [4mTerm[0m is a list with zero or more elements;
  otherwise, returns [4mfalse[0m.

[1mExamples[0m

    1> is_list({a,b,c}).
    false
    2> is_list([]).
    true
    3> is_list([1]).
    true
    4> is_list([1,2]).
    true
    5> is_list([1,2|3]).
    true
