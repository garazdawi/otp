[1m  byte_size(Bitstring)[0m

  Returns an integer that is the number of bytes needed to contain [4m[0m
  [4mBitstring[0m.

  If the number of bits in [4mBitstring[0m is not a multiple of 8, the
  result is rounded up.

[1mExamples[0m

    1> byte_size(<<433:16,3:3>>).
    3
    2> byte_size(<<1,2,3,4>>).
    4
