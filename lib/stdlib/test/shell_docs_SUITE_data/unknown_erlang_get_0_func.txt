[1m  get()[0m

  Returns the process dictionary as a list of [4m{Key, Val}[0m tuples.
  The items in the returned list can be in any order.

  For example:

    1> put(key1, merry).
    2> put(key2, lambs).
    3> put(key3, {are, playing}).
    4> lists:sort(get()).
    [{key1,merry},{key2,lambs},{key3,{are,playing}}]
    5> erase().
    6> get().
    []
