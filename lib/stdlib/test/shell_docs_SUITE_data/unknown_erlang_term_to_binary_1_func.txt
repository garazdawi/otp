[1m  term_to_binary(Term)[0m

  Returns a binary data object that is the result of encoding [4mTerm[0m
  according to the Erlang external term format.

  This can be used for various purposes, such as efficiently writing
  a term to a file or sending an Erlang term through a communication
  channel not supported by distributed Erlang.

  See also [4mbinary_to_term/1[0m.

  [4mNote[0m

    There is no guarantee that this function will always return
    the same encoded representation for the same term.

[1mExamples[0m

    1> Bin = term_to_binary(hello).
    <<131,119,5,104,101,108,108,111>>
    2> hello = binary_to_term(Bin).
    hello
