[1m  tl(List)[0m

  Returns the tail of [4mList[0m, which is the list without its first
  element.

  It works with improper lists.

[1mExamples[0m

    1> tl([geesties, guilies, beasties]).
    [guilies, beasties]
    2> tl([geesties]).
    []
    3> tl([geesties, guilies, beasties | improper_end]).
    [guilies, beasties | improper_end]
    4> tl([geesties | improper_end]).
    improper_end
    5> tl([]).

         in function  tl/1
            called as tl([])
            *** argument 1: not a nonempty list

  Failure: [4mbadarg[0m if [4mList[0m is an empty list [4m[][0m.
