[1m  is_bitstring(Term)[0m

  Returns [4mtrue[0m if [4mTerm[0m is a bitstring (including a binary);
  otherwise, returns [4mfalse[0m.

[1mExamples[0m

    1> is_bitstring(42).
    false
    2> is_bitstring(<<1,2,3>>).
    true
    3> is_bitstring(<<7:12>>).
    true
