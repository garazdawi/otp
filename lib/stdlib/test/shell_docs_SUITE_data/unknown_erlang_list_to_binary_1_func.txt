[1m  list_to_binary(IoList)[0m

  Returns a binary made from the integers and binaries in [4mIoList[0m.

[1mExamples[0m

    1> Bin1 = <<1,2,3>>.
    <<1,2,3>>
    2> Bin2 = <<4,5>>.
    <<4,5>>
    3> Bin3 = <<6>>.
    <<6>>
    4> list_to_binary([Bin1,1,[2,3,Bin2],4|Bin3]).
    <<1,2,3,1,2,3,4,5,4,6>>
