[1m  float_to_binary(Float)[0m

[4mSince[0m:

  There is no documentation for float_to_binary(Float,
  [{scientific, 20}])

[1m  float_to_binary(Float, Options)[0m

[4mSince[0m:
  OTP R16B

  Returns a binary corresponding to the text representation of [4m[0m
  [4mFloat[0m using fixed decimal point formatting.

  [4mOptions[0m behaves in the same way as [4mfloat_to_list/2[0m.

[1mExamples[0m

    1> float_to_binary(7.12, [{decimals, 4}]).
    <<"7.1200">>
    2> float_to_binary(7.12, [{decimals, 4}, compact]).
    <<"7.12">>
    3> float_to_binary(7.12, [{scientific, 3}]).
    <<"7.120e+00">>
    4> float_to_binary(7.12, [short]).
    <<"7.12">>
    5> float_to_binary(0.1+0.2, [short]).
    <<"0.30000000000000004">>
    6> float_to_binary(0.1+0.2).
    <<"3.00000000000000044409e-01">>
