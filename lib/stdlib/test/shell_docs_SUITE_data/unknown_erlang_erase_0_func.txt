[1m  erase()[0m

  Returns the process dictionary and deletes it.

  For example:

    1> put(key1, {1, 2, 3}).
    2> put(key2, [a, b, c]).
    3> lists:sort(erase()).
    [{key1,{1,2,3}},{key2,[a,b,c]}]
