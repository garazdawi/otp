[1m  get_keys(Val)[0m

  Returns a list of keys that are associated with the value [4mVal[0m in
  the process dictionary. The items in the returned list can be in
  any order.

  For example:

    1> put(allosaurus, '🦕').
    2> put(brachiosaurus, '🦕').
    3> put(carnotaurus, '🦕').
    4> put(diplodocus, '🦕').
    5> put(euoplocephalus, '🦕').
    6> put(fox, '🦊').
    7> lists:sort(get_keys('🦕')).
    [allosaurus,brachiosaurus,carnotaurus,diplodocus,euoplocephalus]
    8> erase().
    9> get_keys('🦕').
    []
