[1m  is_number(Term)[0m

  Returns [4mtrue[0m if [4mTerm[0m is an integer or a floating point number;
  otherwise, returns [4mfalse[0m.

[1mExamples[0m

    1> is_number(10.0).
    true
    2> is_number(7).
    true
    3> is_number(zero).
    false
