[1m  get_keys()[0m

[4mSince[0m:
  OTP 18.0

  Returns a list of all keys present in the process dictionary. The
  items in the returned list can be in any order.

  For example:

    1> put(dog, '🐶').
    2> put(cow, '🐄').
    3> put(lamb, '🐑').
    4> lists:sort(get_keys()).
    [cow,dog,lamb]
    5> erase().
    6> get_keys().
    []
